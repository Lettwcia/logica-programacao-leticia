# Aula 01 — Lógica de Programação

1. Definição de algoritmo:
Algoritmo é como instruções do passo a passo para chegar em um resultado

2. O que é entrada, processamento e saída:
Entrada é os dados que vão ser usados processamento é o meio de como vai ser feito e saída é o resultado

3. Exemplo do dia a dia em passos:
(ex: fazer café / escovar os dentes / pedir comida)
Trocar lâmpada
1.Desligar a luz
2.Pegar a cadeira/escada
3.Subir na cadeira/escada e chegar perto da lâmpada
4.Girar a lâmpada no sentido anti-horário até sair
5.Pegar lâmpada nova(mesmo tipo)
6.Encaixar a lâmpada nova
7.Girar pra direita até travar 
8.Descer da escada/cadeiraa
9.Ligar a luz para testar
10.Verificar se está funcionando

4. 5 palavras-chave da aula:
- Versionamento: organização de várias versões do mesmo código
- GIT: Sistema de versionamento
- VS CODE: IDE ambiente onde vai ser trabalhado os códigos
- Hardware e Software: Tudo o que se toca no computador e software o que se vê 
- Pseudocódigo : Algoritmo mais técnico, como rascunho antes de codar
